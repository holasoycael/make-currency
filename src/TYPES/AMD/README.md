# Armenian Dram (AMD)

The **Dram** is the official currency of Armenia. It was first introduced in 1993, replacing the Russian ruble.

---

## 📌 General Information

| Property          | Detail                                                                     |
| :---------------- | :------------------------------------------------------------------------- |
| **Country**       | 🇦🇲 Armenia                                                                 |
| **Symbol**        | `֏`                                                                        |
| **ISO Code**      | `AMD`                                                                      |
| **Currency Unit** | Dram (plural: Dram)                                                        |
| **Subunit**       | Luma (plural: Luma; 1 Dram = 100 Luma - _no longer in active circulation_) |
| **Central Bank**  | Central Bank of Armenia                                                    |

---

## 🎨 Formatting

The Armenian Dram is typically formatted using a comma (`,`) as the decimal separator and a space (` `) as the thousands separator. The currency symbol `֏` is placed after the numeric value, separated by a space.

### Examples

- `0,05 ֏`
- `1,00 ֏`
- `150,75 ֏`
- `1 250,00 ֏`
- `25 000,99 ֏`
