## `CONFIGURE({ money })`

Sets the global default currency for all formatting and parsing functions.

### Signature

<Coding.ApiConfigureSig />

### Parameters

<Table
  columns={configureParamsCols}
  rows={configureParamsRows}
  data-testid="api-configure__table--parameters"
/>

### Example

<Coding.ApiConfigureEx />

> **⚠️ Important:** `CONFIGURE` sets a global singleton state. In SSR or multi-tenant environments, be careful with global state as it persists across requests.
>
> **💡 Default:** If `CONFIGURE` is never called, the global default is `TYPES.USD` (US Dollar).
