    <h2>currency(floatValue, options?)</h2>
    <p>Formats a numeric value into a locale-aware currency string</p>
  </div>
  <div className="api-fn-body">
### Signature
```ts
function currency(floatValue: number, options?: TCurrencyOptions | 'INPUT'): string
```
### Parameters

      <td><code>options</code></td>

### Options (TCurrencyOptions)
<table className="api-table">
  <thead>
    <tr>
      <th>Option</th>

      <th>Description</th>
    </tr>

    <tr>
      <td><code>isEmpty</code></td>

    </tr>
    <tr>
      <td><code>money</code></td>
      <td><code>TExchange</code></td>
      <td><code>TYPES.USD</code></td>
      <td>Currency configuration (locale + code + transforms)</td>
    </tr>
  </tbody>
</table>
### Returns
`string` — The formatted currency string.
### Examples
```ts
import { currency, TYPES } from 'make-currency'
// Default (USD)
currency(1234.56)                                      // "$1,234.56"
// Without symbol
currency(1234.56, { symbol: false })                   // "1,234.56"
// With a specific currency
currency(1234.56, { money: TYPES.BRL })                // "R$ 1.234,56"
currency(1234.56, { symbol: false, money: TYPES.BRL }) // "1.234,56"
// Empty string when value is zero
currency(0, { isEmpty: true })                         // ""
currency(0, { isEmpty: false })                        // "$0.00"
// INPUT shortcut — symbol: false, isEmpty: true, uses global currency
currency(1234.56, 'INPUT')                             // "1,234.56"
```
<div className="api-note">
  <strong>💡 'INPUT' shortcut:</strong> Pass <code>'INPUT'</code> as options to use the global currency (set via <code>CONFIGURE</code>) with <code>symbol: false</code> and <code>isEmpty: true</code>. Ideal for controlled input components.
</div>

### Return Type (ICurrencyProps)
```ts
interface ICurrencyProps {
  floatValue: number   // The numeric value (e.g. 1234.56)
  stringValue: string  // The formatted string without symbol (e.g. '1,234.56')
}
```
### Examples
```ts
import { currencyFn, TYPES } from 'make-currency'
currencyFn('123456')
// { floatValue: 1234.56, stringValue: '1,234.56' }
currencyFn('1')
// { floatValue: 0.01, stringValue: '0.01' }
currencyFn('')
// { floatValue: 0, stringValue: '' }
// With BRL
currencyFn('123456', { money: TYPES.BRL })
// { floatValue: 1234.56, stringValue: '1.234,56' }
```

    <h2>{'CONFIGURE({ money })'}</h2>
    <p>Sets the global default currency for all formatting and parsing functions</p>
  </div>
  <div className="api-fn-body">
### Signature
```ts
function CONFIGURE({ money }: CONFIGUREProps): void
```
### Parameters
<table className="api-table">
  <thead>
    <tr>
      <th>Parameter</th>
      <th>Type</th>
      <th>Required</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>money</code></td>
      <td><code>TExchange</code></td>
      <td>Yes</td>
      <td>The currency configuration to set as global default</td>
    </tr>
  </tbody>
</table>
### Example
```ts
import { CONFIGURE, TYPES, currency, currencyFn } from 'make-currency'
// Set BRL as the global currency
CONFIGURE({ money: TYPES.BRL })
// Now all calls without `money` use BRL
currency(1234.56)          // "R$ 1.234,56"
currencyFn('123456')       // { floatValue: 1234.56, stringValue: '1.234,56' }
// You can still override per-call
currency(1234.56, { money: TYPES.USD }) // "$1,234.56"
```
<div className="api-warning">
  <strong>⚠️ Important:</strong> <code>CONFIGURE</code> sets a global singleton state. In SSR or multi-tenant environments, be careful with global state as it persists across requests.
</div>
<div className="api-note">
  <strong>💡 Default:</strong> If <code>CONFIGURE</code> is never called, the global default is <code>TYPES.USD</code> (US Dollar).
</div>
  </div>

    <p>Object containing all 139 currency configurations keyed by ISO 4217 code</p>

### TExchange Structure
Each currency in `TYPES` follows the `TExchange` interface:
```ts
type TExchange = {
  lang: TLanguage           // Locale string (e.g. 'pt-BR', 'en-US')
  currency: TCurrency       // ISO 4217 code (e.g. 'BRL', 'USD')
  removePrefix: (value: string) => string  // Removes the symbol from formatted output
  replaceSymbol?: (value: string) => string // Optional custom symbol replacement
}
```
### Usage
```ts
import { TYPES } from 'make-currency'
TYPES.USD  // { lang: 'en-US', currency: 'USD', removePrefix: ... }
TYPES.BRL  // { lang: 'pt-BR', currency: 'BRL', removePrefix: ... }
TYPES.EUR  // { lang: 'pt-PT', currency: 'EUR', removePrefix: ... }
TYPES.JPY  // { lang: 'en-US', currency: 'USD', removePrefix: ..., replaceSymbol: ... }
```
